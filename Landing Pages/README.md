# Landing Page - Producto Increíble

Una landing page moderna y responsiva para promocionar un producto excepcional, construida con HTML, CSS y JavaScript.

## 📋 Descripción

Esta landing page presenta un producto de alta calidad con un diseño elegante y moderno. Incluye secciones para hero, beneficios, imagen del producto y formulario de contacto para capturar leads.

## ✨ Características

- **Diseño Responsivo**: Se adapta perfectamente a dispositivos móviles y desktop
- **Interfaz Moderna**: Diseño oscuro con elementos visuales atractivos
- **Formulario Interactivo**: Captura de información de contacto con validación
- **Secciones Organizadas**:
  - Hero con imagen de fondo
  - Beneficios destacados
  - Imagen del producto
  - Formulario de contacto

## 🛠️ Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos modernos con Grid y Flexbox
- **JavaScript**: Interactividad del formulario
- **Google Fonts**: Tipografía Arial

## 🚀 Instalación y Uso

1. **Clona el repositorio**:
   ```bash
   git clone https://github.com/DeprivedVisit/Garett-Barrantes-Benavides.git
   cd "Landing Pages"
   ```

2. **Abre el archivo**:
   - Abre `index.html` en tu navegador web
   - O usa un servidor local como Live Server en VS Code

## 📁 Estructura del Proyecto

```
Landing Pages/
├── index.html          # Archivo principal HTML
├── style.css           # Estilos CSS
├── script.js           # JavaScript para el formulario
└── README.md           # Este archivo
```

## 🎨 Personalización

### Cambiar el contenido:
- Edita el texto en `index.html`
- Modifica los estilos en `style.css`

### Cambiar la imagen del producto:
- Reemplaza la URL en `index.html` línea 42
- O sube tu propia imagen y actualiza la ruta

### Modificar colores:
- Variables principales en `style.css`:
  - Fondo: `background:#0f0f0f;`
  - Tarjetas: `background:#1a1a1a;`
  - Inputs: `background:#333;`
